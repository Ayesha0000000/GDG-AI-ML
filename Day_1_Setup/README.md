# Day 1: Environment Setup

## Objective

To set up development environment using WSL, Ubuntu, and VS Code.

---

## Tasks Performed

* Checked Windows version using `winver`
* Enabled virtualization from Task Manager
* Installed WSL using:
  wsl --install
* Installed Ubuntu and created username/password
* Verified WSL version using:
  wsl -l -v
* Updated packages using:
  sudo apt update && sudo apt upgrade -y
* Installed Visual Studio Code
* Installed WSL extension in VS Code
* Connected VS Code with WSL using:
  code .

---

## Why This is Important

This setup allows us to use Linux inside Windows.
It is important because many development tools work better in Linux environment.
VS Code integration makes coding easier and more efficient.

---

## What I Learned

I learned how to install WSL and connect it with VS Code.
I also understood how Linux commands work and how to manage environment.

---

## Result

Successfully set up development environment with WSL and VS Code.
